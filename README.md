# Allstate Claims Severity

Machine learning solution for the Kaggle Allstate Claims Severity competition.

## Project Overview

This project predicts insurance claim severity using CatBoost regression and feature engineering techniques.

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- CatBoost

## Project Structure

- `prepare.py` - data loading and preprocessing
- `main.py` - model training and submission generation
- `notebooks/` - exploratory data analysis

## Results

Model trained on log-transformed target (`loss`) using CatBoostRegressor.
